velocity = clamp( (recent_avg − baseline_avg) / baseline_avg × 5, 0, 10 )
composite = Σ velocity[s] × weight[s]  // news=0.28, finance=0.32, site=0.22, social=0.18
if active_sources ≥ 3: composite ×= 1.4 // cross-domain amplification
alert = score ≥ 6.5 OR z-score > 2.5 OR confidence ≥ 0.80
📈 Trajectory Slope
Linear regression across velocity history — positive = accelerating toward event
🔔 Anomaly Detection
Z-score analysis: if latest score is 2.5σ above history → immediate alert

Claude Sonnet receives velocity + signals → natural-language prediction with probability
